Before you continue...

Our checklist for a smooth start:

  1. Have an ERC-20 (Ethereum) compatible wallet ready. Not an exchange wallet.
    Your Gunbot license will be registered to this wallet address.

    Sounds complicated? It just takes a few minutes if you follow these instructions.

  2. Create an API key for at least one exchange you want to use Gunbot on.
    You can update your API keys at any later time.
    Use fictional placeholders if you just aren't ready to create your API key(s) yet.

    Not sure how to do this? Follow this guide.
