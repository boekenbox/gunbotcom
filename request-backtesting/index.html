A few tips ...

Due to time zone differences, please allow for up to 24h activation time.

  1. Use an existing username on tradingview.com
    Access to the backtesting script is coupled to your username. Changing the registered username is possible only once.
  2. Scripts will show in the invite only category
    After access has been granted, you'll find the backtesting scripts in the "invite only scripts" category when adding an indicator to a chart on tradingview.com
