Tips for using the demo

You cannot make changes to the demo system. They are live trading bot instances, but only meant for demonstration purposes. To see data for all active trading pairs, go to the dashboard where you can get detailed info about all current trading targets. Click "PNL analysis" on the chart to get detailed profit and loss overviews, for all and for individual trading pairs. In case the bot looks like it has stopped, push the "Start Core" button on the settings page. You'll soon see data appearing on the dashboard and chart.
