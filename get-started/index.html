Frequently asked questions

What is GUNTHY token?

Gunbot uses a system to validate license ownership through the blockchain. To do this, it uses a utility token called "GUNTHY" that runs on the Ethereum blockchain. The end goal of this system is to not depend on a central license server. An added benefit is that you can even sell your license.

I'm new to this, will you help?

Sure! We know this software can be a bit intimidating to start with. That's exactly why we run our Awesome Gunbot support community.

Is Gunbot difficult to learn?

Gunbot is easy to start with, and admittedly hard to fully master - as with any complex software project. Be prepared to learn something new every week, it's fun though!
